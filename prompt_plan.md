# 구현 계획: 프리미엄 격상 P0 — 첫인상 (확정 2026-06-16)

> 근거: docs/premium-audit.md · CTO 위임 승인

## 목표
- ① 어시스턴트 넛지가 첫 페인트에 제품/CTA를 가리지 않게(세션 1회 유지).
- ② 제품 상세 데스크톱 히어로 여백 과다·구도 부재 개선. 모바일 회귀 0.
- 제약: paper/ink/gold·타이포 유지, 모바일 폭 회귀 금지, 변경 최소.

## Phase 1 — 넛지 등장 타이밍 (components/CustomerAssistant.tsx, LOW)
- `ready` 상태 추가. 첫 화면 60% 스크롤 또는 6초 경과 중 먼저 도달 시 `ready=true`.
- 렌더 조건 `!open && showNudge` → `!open && showNudge && ready`.
- 타이머/스크롤 리스너 클린업. 첫 페인트엔 플로팅 버튼만.
- 검증: 첫 로드 스크린샷에 넛지 없음 → 스크롤/지연 후 등장.

## Phase 2 — 데스크톱 히어로 lg 2단 (app/products/[id]/page.tsx, MEDIUM)
- `lg:`(≥1024px) 전용 비대칭 2단: 좌 대형 제품 이미지 + 우 카테고리·제품명·태그라인·가격·CTA(좌측정렬).
- 1024px 미만은 기존 마크업/순서 그대로(모바일 DOM 불변).
- `#configure` 앵커·스티키·옵션 패널 흐름 유지.
- 대안(보수적): 구조 유지, 여백만 축소 — 채택 안 함(권장안으로 진행).

## 위험
- MEDIUM: Phase 2가 앵커/스티키/옵션 흐름 건드릴 수 있음 → lg 분기로 격리, 모바일 불변 보장.
- LOW: Phase 1 리스너 누수 → 클린업.
- 회귀: 모바일 375 scrollWidth=375·오버플로 0 재확인.
