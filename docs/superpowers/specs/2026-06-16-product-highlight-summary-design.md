# 제품 상세페이지 요약 하이라이트 — 설계

작성일: 2026-06-16

## 결론

제품 상세페이지 히어로의 CTA 버튼 아래 빈 공간에, **"0.01%" 브랜드 선언 한 줄 + 무선(無線) 스펙시트**를 추가한다. 효능 표현 없이 성분·공정·수치·사육방식 등 **검증된 사실만** 노출해 프리미엄 차별성을 전달한다. 우유·요거트 4개 SKU 전체 적용.

## 배경 / 목적

- 현재 상세페이지는 가격·버튼 아래가 비어 있고, 제품의 차별점(요약)이 본문 스크롤 전에는 보이지 않는다.
- 레퍼런스(타 페이지의 이모지 불릿 요약)처럼 "스캔 한 번에 차별점"을 주되, 글머리 이모지는 배제하고 **애플 제품페이지 어법(타이포·여백·숫자)**으로 격상한다.
- 기존 히어로 구조(eyebrow·제목·태그라인·가격·배지·버튼)는 **한 줄도 변경하지 않는다.** 버튼 아래에 섹션을 **추가**만 한다.

## 디자인 결정 (확정)

- **레이아웃**: 히어로 정보 컬럼 끝(버튼 다음). 상단 얇은 구분선으로 분리.
- **선언(kicker)**: 명조, 2줄 대칭형. 우유 "0.01%의 소로, **0.01%**를 위해." / 요거트 "0.01%의 우유로, **0.01%**를 위해." 두 번째 `0.01%`만 골드 강조.
- **스펙시트**: 항목 사이 **실선 없음**(여백으로만 구분). 좌측 소형 대문자 라벨 + 우측 값. 굵은 키워드(잉크)와 숫자(명조 강조)로 위계.
- **숫자 강조**: `75℃` `20초` `2:1` `7.2억` `12시간` `15%/16%` `3.5g` 등 핵심 수치는 명조로 살짝 키워 노출.
- **출처 한 줄**: 공인 분석 접수번호 표기(신뢰 단서).

## 카피 (효능 표현 없음 · 규제 안전선 준수)

### 우유 (milk-180 / milk-750 공유)
- 선언: 0.01%의 소로, **0.01%**를 위해.
- 원유: **A2 저지.** A2 베타카제인 단백질만.
- 사육: **헤이밀크.** 풀과 건초. 사일리지는 없습니다.
- 공정: **75℃**에서 **20초**. 최소한의 균질.
- 오메가: 6:3, **2:1**의 균형. *(당사 분석 기준)*
- 칼슘: 하루 권장의 **15%**. *(100g당 104.9mg)*
- 원산지: **100% 국내산.**
- 출처: 공인 영양성분 분석 · 26-06-BR0114

### 요거트 (yogurt-180 / yogurt-500 공유)
- 선언: 0.01%의 우유로, **0.01%**를 위해.
- 원유: **그 우유, 그대로.** 0.01%의 A2 저지.
- 발효: 유산균과 **12시간**. 그뿐입니다.
- 유산균: **1g당 7.2억** 마리. *(공인 시험성적서)*
- 당류: **무설탕** · **3.5g**. *(발효로 원유 5.9g → 3.5g, 100g당)*
- 칼슘: 하루 권장의 **16%**. *(100g당 108.7mg)*
- 출처: 공인 영양성분 분석 · 26-06-BR0115 · 무증점제

## 규제 가드레일 (식품 등의 표시·광고에 관한 법률)

- 일반식품(우유=축산물, 요거트=발효유)이므로 **질병 예방·치료·기능성(소화·장건강·면역) 암시 금지.**
- 허용: 성분·품종·사육·살균·등급·수치 등 **객관적 사실 서술**. 위 카피는 전부 여기에 해당.
- `2:1` 오메가 비율은 **"당사 분석 기준"** 출처를 반드시 병기.
- "농후발효유" 등 **미등록 등급명 사용 금지**(현재 등록=발효유). 요거트 식품유형 호상/드링킹 표기는 법정 라벨이므로 본 작업에서 **변경하지 않음**(차후 사실 확인).
