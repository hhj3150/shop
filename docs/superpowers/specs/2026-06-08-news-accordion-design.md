# 홈 목장소식 제목 표시 + 인라인 펼침 — 설계

- 날짜: 2026-06-08
- 상태: 승인
- 범위: 홈 `components/NewsBand.tsx` 표시 방식만 변경

## 배경

홈 목장소식이 카드에 본문 일부(2문단)까지 길게 노출돼 화면을 많이 차지한다.
제목만 보이고, 클릭하면 전체 내용을 보도록 바꾼다.

## 결정 (승인됨)

- **인라인 펼침(아코디언)** 방식. 별도 상세 라우트·모달 없음.
- **단일 컬럼 목록** 레이아웃(기존 2열 카드 그리드 → 세로 목록).
- 관리자 CRUD(`/admin/news`)는 이미 충분 → **변경 없음**. 데이터·스키마·라우트 변경 없음.

## 동작

- 기본(접힘): 각 소식은 **날짜(+영상 여부) + 제목**만 표시. 우측에 펼침 셰브론.
- 행 클릭 → 그 자리에서 **전체 펼침**: 대표사진 또는 유튜브 임베드 + 본문 전체 문단.
  다시 클릭하면 접힘.
- 항목별 독립 토글, 첫 로드는 전부 접힘.
- 접근성: 토글 버튼 `aria-expanded`, 셰브론은 `aria-hidden`.

## 데이터

- 변경 없음: `news` 테이블에서 `published=true` 최신 4개 조회(기존과 동일).

## 비목표(YAGNI)

- 상세 페이지/모달, 공유 링크, 예약 게시, 정렬 변경, 관리자 기능 추가.

## 검증

- `tsc` 0 + 전체 `vitest`(순수 로직 `lib/news` 회귀 없음) + Netlify `next build`.
- 육안: 접힘 상태 제목만, 클릭 시 펼침/접힘 동작.
